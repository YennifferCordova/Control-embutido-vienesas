# Control de Proceso — Vienesas 🌭

App de cartas de control estadístico conectada a SharePoint.

## Cómo publicar gratis en Streamlit Cloud

1. Sube esta carpeta a un repositorio en GitHub (github.com)
2. Ve a share.streamlit.io → "New app"
3. Conecta tu cuenta de GitHub → selecciona el repositorio
4. Haz clic en Deploy → en 2 minutos tienes el enlace

## Cómo usar la app

1. Abre el enlace
2. En el panel izquierdo ingresa:
   - URL del sitio SharePoint (ej: https://tuempresa.sharepoint.com/sites/Calidad)
   - Correo corporativo
   - Contraseña
3. Haz clic en "Conectar y cargar datos"

## Cartas disponibles

- Carta X̄ — Largo vienesa (VEMAG 1 y VEMAG 2)
- Carta X̄ — Torsiones (VEMAG 1 y VEMAG 2)
- Carta p — % de rechazo por muestreo
- Carta C — Stick rotos con barras apiladas por tipo de tripa
- Historial con descarga CSV
